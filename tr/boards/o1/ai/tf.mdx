## 1. Ortamın Hazırlanması

Gerekli çalışma ortamının oluşturulabilmesi için öncelikle GitHub üzerinden examples deposu indirilmelidir.

```bash
git clone https://github.com/t3gemstone/examples.git
```

Depo indirildikten sonra, bağımlılıkların kurulumu için ./setup.sh betiği çalıştırılmalıdır.

```bash
./setup.sh
```

Kurulum tamamlandıktan sonra gerekli araçların otomatik olarak yüklenebilmesi için jetify-devbox kabuğu etkinleştirilmelidir.

```bash
devbox shell
```

Projeleri cross-compile olarak derleyebilmek için gerekli toolchain’in indirilmesi gerekmektedir.

```bash
task fetch
```

Projede yer alan kaynak kodlarda değişiklik yapmak isterseniz, ilgili dosyalara git ile clone yaptığınız ```examples``` klasörünün içindeki ```camera-ai-accelerator``` dizini üzerinden ulaşabilirsiniz.


Ardından proje derleme işlemi gerçekleştirilebilir.

```bash
PROJECT=camera-ai-accelerator task clean build 
```
## 2. Gerekli Malzemelerin Hazırlanması

<Tooltip tip="CSI Kamera bağlanması durumunda overlay eklenmesi gerekmektedir." cta="CSI Kamera eklemeye gözat" href="/tr/boards/o1/peripherals/camera">Kamera</Tooltip> bağlantımızı yaptıktan sonra T3 Gemstone Obsidyen geliştirme kartına güç vererek kartı çalıştıralım. (Gem-Imager ile işletim sistemini kurduğunuz varsayılmıştır.)

Derlenen projenin ve model dosyalarının geliştirme kartına aktarılması yapılmalıdır. Aşağıda SCP üzerinden derlenmiş ```camera-ai-accelerator``` dosyası aktarılmıştır.

```bash
scp build/examples/camera-ai-accelerator/camera-ai-accelerator gemstone@192.168.7.2:/home/gemstone/
```

## 3. Projenin Çalıştırılması

Projeyi çalıştırırken bir model dosyası belirtmeniz zorunludur. ```<modelinizin-yolu>``` alanına, hazır bir modeli veya derlemiş olduğunuz modelin yolunu girmeniz gerekmektedir.

Varsayılan olarak proje, CSI üzerinden bağlanmış IMX219 kamera ile çalışacak şekilde yapılandırılmıştır. Farklı bir kamera sensörü kullanmanız durumunda kaynak kodunda gerekli değişiklikleri yapmanız gerekmektedir.
Desteklenen kamera sensörlerinin listesi ```/opt/imaging/``` klasörü içerisinde bulunmaktadır.
Farklı bir kamera ile çalışabilmek için GST Pipeline’ı bu dosyalara göre güncellemeniz yeterlidir.

USB üzerinden bağlanan kameraları kullanmak için ise, projeyi çalıştırırken ```--device=/dev/USBx``` argümanınına kameranın bağlı olduğu USB cihazını eklemeniz yeterlidir. Bu sayede ek bir yapılandırma yapmadan USB cihazdan gelen görüntüyü projede kullanabilirsiniz.
```bash
chmod +x camera-ai-accelerator
./camera-ai-accelerator --modeldir=<modelinizin-yolu>

```
