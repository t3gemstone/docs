<Tip>C7-based MCU projects using the DSPLIB library operate by utilizing the DSP accelerator unit within the processor. This allows fundamental DSP operations such as filtering, convolution, correlation, and similar tasks to be accelerated in hardware, resulting in high performance for real-time applications.</Tip>

This document describes the integration steps and sample project for the Texas Instruments DSPLIB library in MCU projects developed on the T3 Gemstone O1 development board.
DSPLIB is a software library optimized for the C7x architecture that contains low-level Digital Signal Processing (DSP) functions.
It is designed to perform filtering, correlation, convolution, transformations, and other basic DSP operations with high efficiency and is used specifically to reduce processor load in real-time DSP/MCU applications.
